# ITECHIE — Freelance Engineering Collective

A production-ready landing page for iTechie, built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- **Next.js** (App Router)
- **Tailwind CSS v4**
- **Framer Motion**
- **shadcn/ui**
- **TypeScript**
